Install on Firestick — the easy way

  1. On your Firestick: Settings → My Fire TV → Developer Options → turn on Install unknown apps (allow it for Downloader).
  2. Install Downloader from the Amazon Appstore — it's free and official.
  3. Open Downloader, click the URL box, and enter the code above.
  4. It downloads the APK and asks to install. Hit Install, then Open.
  5. On first launch, paste in your M3U playlist URL. Guide data and weather are optional.
Won't work on the newest Firesticks. Amazon's latest sticks run Vega OS, which blocks sideloading and the Downloader app entirely. If your stick is a Vega model, you'll need an older Fire TV or an Android/Google TV box instead.

Install by ADB (if you prefer)

  1. Enable ADB Debugging in Developer Options.
  2. Grab your stick's IP: Settings → My Fire TV → About → Network.
  3. From your computer, on the same Wi-Fi:
    adb connect YOUR_IP:5555  then  adb install PrestonsPlayer.apk
